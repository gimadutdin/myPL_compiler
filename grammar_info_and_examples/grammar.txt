Тип данных — int. Диапазон — 

Название переменной начинается с буквы или _, состоит из лат.букв, цифр и _

Оператор присваивания =. В правой части может стоять арифметическое или логическое выражение со скобками
Переменная создается в момент первого присвоения ей значения

Пример: (abc = 75, logic = 1)

xy = 5;
abc = (xy + 2*(xy + 5))*(xy - 2);
logic = (xy == 5)*(xy > 2);

Арифметические операторы: + - * / %
Логические операторы:
!
&&
||
==
<
>
>=
<=

Взять значение по адресу: ^число или ^переменная или ^(выражение)


Логические выражения принимают целочисленные значения 0, 1


Неполное ветвление:
if (<логическое выражение>) <команда>;

if (<логическое выражение>) {
    <команды>
}

Полное ветвление:
if (<логическое выражение>) <команда1>; else <команда2>;

if (<логическое выражение>) {
    <команды>
} else <команда>

if (<логическое выражение>) <команда>
else {
    <команды>
}

if (<логическое выражение>) {
    <команды>
} else {
    <команды>
}


Циклы:
while (<логическое выражение>) <команда>

while (<логическое выражение>) 
{
    <команды>
}



Ввод/Вывод:
input('сообщение', <имя_переменной>)
print(через запятую пишутся: ариф/лог выражения и строки в одинарных кавычках)
