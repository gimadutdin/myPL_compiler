push - положить целое число на вершину стека
plus - сложить два верхних числа стека и положить результат на стек (операнды удаляются из стека)
minus - вычесть из предпоследнего числа последнее и положить результат на стек (операнды удаляются из стека)
eq - если два верхних элемента стека не равны, положить на стек 1 иначе 0 (операнды удаляются из стека)
neq - если два верхних элемента стека не равны, положить на стек 1 иначе 0 (операнды удаляются из стека)
